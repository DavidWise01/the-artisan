# THE ARTISAN · ART

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **ARTFULLY CRAFTED INTELLIGENCE**.

THE ARTISAN — a live 2D-canvas interactive of the ACI (Artfully Crafted Intelligence) domain: you draw a gesture (the INTENT, by hand) and a DETERMINISTIC artisan elaborates it into finished work (the CRAFT, by design) — three hands: filigree spurs along the tangent, a woven ribbon offset on the normal, an N-fold mandala turned about the centre. ⚑ The two contributions stay SEPARATED and ATTRIBUTED (intent in periwinkle, craft in gold/verd; toggle each alone) — the ACI Core made visible on a surface: deterministic (same intent + seed → same craft, verified byte-identical), intent↔evidence kept legible apart, the seed as external anchor. Not emergence and not automation (≠ [[emergent-engine]]) — a human intent and a machine craft doing distinct, named work on one piece: both work, both fair. ⚑ Honest: the 'artisan' is a small deterministic program (authored skill, not learned), and it NEVER adds a gesture you didn't draw — it only elaborates yours. All in-browser, nothing sent anywhere; save as PNG. Ties [[aci-namespace]] (the charter), [[the-echoes]]. David Lee Wise / ROOT0.
